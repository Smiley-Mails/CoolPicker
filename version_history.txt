Version 2.3 February 2009 DB
	Fixed a bug in the Mixer with gamma correction enabled.
	Fixed picking from the screen when the values are set up to 0-100.

	Fixed a couple of bugs with floating point colors.

Version 2.0.1 Nov 20 2005 CJ
	Added functionality to pick a color from a sampled screen pixel
	anywhere on the screen. This was done in two parts. First adding
	a button control in the dialog editor, and adding code for when it
	gets pressed. When it get's pressed, it changes the cursor, does 
	some other stuff and returns. The second part is handled when the 
	user clicks the mouse. Then it sames the window, the hdc, and finally
	asks for the pixel info. Once it does that, it restores the cursor
	back to the old cursor.
